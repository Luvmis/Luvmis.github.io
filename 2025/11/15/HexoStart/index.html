Hexo using

在根目录用终端选择框架初始化文章

hexo new [layout] [name]

更新博客

hexo clean && hexo g && hexo d

图床搭建

PicGo + aliyun OSS + Typora

aliyun OSS配置：

  1. 进入控制台，搜索对象存储 OSS，创建Bucket（名字有用）
  2. 鼠标放在头像进入Accesskey获取AccessKey ID,AccessKey Secret（选定的地区后面有用）
  3. 进入PicGo在图床设置的阿里云OSS，填写对应配置信息
